UX Linter Extension Installation

Installation Steps:

  1. Click the "Install Extension" button below
  2. Chrome will open the extensions page
  3. Enable "Developer mode" in the top right
  4. Click "Load unpacked"
  5. Select the "src" folder from the downloaded extension
Install Extension
Note: If the button doesn't work, manually open Chrome and go to: chrome://extensions

After Installation:

  • The extension icon will appear in your Chrome toolbar
  • Click the icon on any webpage to see UX violations
  • Use the "Refresh" button to rescan the page
  • Use "Clear Highlights" to remove the violation indicators
